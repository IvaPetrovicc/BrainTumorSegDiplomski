# Brain Tumor Segmentation (Diplomski)

Ovaj projekt je razvijen kao diplomski rad za segmentaciju tumora na MRI snimkama.

Aplikacija se sastoji od:
- FastAPI backenda za treniranje i predikciju modela
- Vue frontend aplikacije za upload slike i prikaz rezultata segmentacije
- YOLO treniranog segmentation modela (`best.pt`)
